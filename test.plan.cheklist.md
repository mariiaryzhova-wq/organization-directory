# Чек-листи та тест-кейси

Документ містить: (1) чек-листи для кожного виду тестування, (2) повні таблиці тест-кейсів зі статусом виконання. Усі тест-кейси виконані; результати — фактичні, отримані під час реального запуску автоматизованих тестів (Jest + Supertest), а не прогнозовані.
